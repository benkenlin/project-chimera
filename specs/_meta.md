## Factory Purpose
Build autonomous AI influencer agents that operate safely in both human and agent social networks.

## Factory Departments:

### 1. Research & Development (R&D)
**Location:** `specs/` folder
**Responsibility:** Create blueprints and instructions
**Output:** Clear, executable specifications

### 2. Manufacturing Line
**Location:** `skills/` folder  
**Responsibility:** Build agent capabilities
**Output:** Reusable skills (functions)

### 3. Quality Control
**Location:** `tests/` folder
**Responsibility:** Ensure everything works correctly
**Output:** Passing tests and safety checks

### 4. Safety & Governance
**Location:** `governance/` folder
**Responsibility:** Enforce rules and ethics
**Output:** Safe, compliant content

### 5. Shipping & Deployment
**Location:** `.github/workflows/`
**Responsibility:** Automated delivery
**Output:** Live, working agents

## Factory Workers:
- **AI Agents:** Follow specifications, use skills, pass tests
- **Human Oversight:** Approve critical decisions, review outputs
- **Automated Systems:** Run tests, check compliance, deploy updates
